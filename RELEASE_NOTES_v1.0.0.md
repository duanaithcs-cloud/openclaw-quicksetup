# OpenClaw QuickSetup v1.0.0

Bản phát hành đầu tiên của bộ cài nhanh OpenClaw cho người mới.

## Điểm nổi bật
- Cài đặt theo luồng rõ ràng: **Nhập thông tin -> Test -> Cài đặt**.
- Hỗ trợ Telegram + Zalo + ProxyPal trong 1 UI.
- Có kiểm tra PASS/FAIL trước khi áp cấu hình.
- Có one-click installer với backup và báo cáo tự kiểm.

## Cách chạy nhanh
1. Chạy: `launchers/46_OpenClaw_QuickSetup_UI.bat`
2. Nhập token + ID.
3. Bấm `Test tất cả`.
4. Bấm `Tạo + Cài đặt ngay (v2)`.
