# Security Policy

## Supported Versions
Hiện tại hỗ trợ nhánh `main` (bản mới nhất).

## Reporting a Vulnerability
Nếu phát hiện lỗ hổng bảo mật:
- Không mở issue công khai chứa token/secret.
- Liên hệ chủ repo trực tiếp để xử lý trước khi public.

## Security Notes
- Dự án không nên lưu token thực trong Git.
- Các file output/cache đã được loại qua `.gitignore`.
