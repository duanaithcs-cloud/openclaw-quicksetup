# Contributing

Cảm ơn bạn đã muốn đóng góp cho OpenClaw QuickSetup.

## Quy tắc nhanh
- Giữ luồng cho người mới thật đơn giản (ít bước, rõ ràng).
- Không đưa secrets/token thật vào code hoặc ảnh chụp.
- Mọi thay đổi cần giữ tương thích với workflow 1-2-3.

## Cách đóng góp
1. Fork repo
2. Tạo branch mới: `feature/<ten-tinh-nang>`
3. Commit rõ ràng, nhỏ gọn
4. Tạo Pull Request mô tả:
   - vấn đề cũ
   - giải pháp mới
   - ảnh trước/sau (nếu thay UI)

## Checklist trước PR
- [ ] Chạy app không lỗi syntax
- [ ] Không có file cache/output rác
- [ ] README cập nhật nếu đổi luồng sử dụng
