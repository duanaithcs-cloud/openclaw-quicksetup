# GitHub Growth Checklist

## Profile Repo Hygiene
- [x] README rõ ràng + dễ chạy
- [x] LICENSE
- [x] CONTRIBUTING
- [x] SECURITY
- [x] CHANGELOG
- [x] Release notes
- [x] CI compile check

## 7-day Growth Sprint

### Day 1
- Publish release v1.1.0
- Ghim repo lên đầu profile
- Cập nhật bio GitHub có link repo

### Day 2
- Đăng post launch Facebook + Telegram cộng đồng
- Nhờ 5-10 bạn đầu tiên dùng thử

### Day 3
- Mở issue template cho bug/feature request
- Trả lời mọi comment trong 24h

### Day 4
- Làm video demo ngắn 30-60s
- Gắn vào README

### Day 5
- Viết bài technical deep-dive về workflow

### Day 6
- Làm bản `v1.2.0` theo phản hồi thật

### Day 7
- Tổng kết changelog + cảm ơn cộng đồng

## KPI tối thiểu tuần đầu
- 30+ stars
- 5+ issues/phản hồi chất lượng
- 2+ PR/community contributions
